# Молитва вдвоём

Небольшой сайт для совместной молитвы с автоматической подгрузкой тропарей дня на текущий день.

## Запуск

```bash
npm start
```

Сайт откроется на `http://127.0.0.1:3000`.

Важно: не открывайте `public/index.html` напрямую из файловой системы. Блок с тропарями работает через локальный endpoint `/api/troparion`, поэтому для него нужен запуск через `npm start`.

## Публикация

GitHub нужен для хранения кода, но для работы сайта онлайн нужен Node-хостинг, потому что тропари подгружаются через серверный endpoint `/api/troparion`.

В проект уже добавлен [render.yaml](/Users/macbook/Documents/Codex/2026-05-29/1-2-3-4-5-6/render.yaml), так что после загрузки на GitHub сайт можно быстро подключить к Render как web service.

## Что внутри

- `public/index.html` — основная структура страницы и молитвенный текст.
- `public/styles.css` — оформление, адаптивность и стили блока тропарей.
- `public/app.js` — загрузка тропарей дня и копирование основного текста.
- `server.js` — статический сервер и прокси `GET /api/troparion`, который парсит страницу дня с Азбуки веры.

## Источник тропарей

Используется церковный календарь сайта Азбука веры:

- [Азбука веры](https://azbyka.ru/days/)
- Формат страницы дня: `https://azbyka.ru/days/YYYY-MM-DD`
